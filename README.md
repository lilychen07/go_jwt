## Introduce

   这是一个基于beego框架搭建的一个小项目，主要功能包含如下：
   
   1. Websocket
   
   2. 微信公众号，小程序
   
   3. jwt相关
   
   4. Redis
   
   5. Mysql
   
   6. ElasticSearch (未集成)
 
 
 

### 快速开始

```
   1、将项目拉到本地，修改conf/app.conf中相关配置

   2、执行 ./run.sh start 即可启动项目

   3、执行 ./run.sh stop 停止运行

   4、执行 ./run.sh restart 重启

```
